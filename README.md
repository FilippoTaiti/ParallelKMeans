Le specifiche hardware utilizzate e il sistema operativo sono riportati nella relazione.
Si può eseguire il programma eseguendo il file main.cpp, dove ci sono i parametri k (numero centroidi/ cluster), n (numero elementi del dataset da utilizzare) e number_of_iterations. Tali parametri possono essere modificati al fine di generare ulteriori test.
Nella cartella Utility vi è l'implementazione di funzioni di utilizzo generale per entrambe le versioni (sequenziale e parallela). 
E' stata utilizzata una MACRO parallel al fine di generare due eseguibili, uno per la versione sequenziale ed uno per la versione parallela. Dunque, se si builda da terminale verranno generati due eseguibili.
